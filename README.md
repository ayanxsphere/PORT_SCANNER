# PORT_SCANNER
pORT‑SCANNER (mini‑scanner) — Lightweight threaded TCP port scanner in Python.
Run: python main.py 127.0.0.1 1 65535 --threads 200 --timeout 0.5.
Note: For educational use only — scan only machines .
